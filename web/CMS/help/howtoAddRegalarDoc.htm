This describes how to add a regular document (that is, not an index page, and not a pdf/word file)
To add a regular document:
  • Click on the Documents link in the left menu.
  • Click the new button
  • This will take you to a screen where you can enter the properties of the new document
  • First of all, select the language you want the document to be in, by clicking one of the flags in the upper right corner
  • Now enter appropriate values in the fields:
  • Type: select the type the document is (see Types and Document in the help menu for a further description on this)
  • Identifier: Enter an identifier, enabling you to identify the document later, this will only be visible by you
  • Priority: enter a priority of the document. default is 100. A higher priority means the document will appear higher on index-pages. If unsure enter 100
  • DONT hit the save button, but keep filling out information below
  • Linktext: enter the text you want to be seen when the document appears on index-pages, etc
  • Format: Mark regular
  • Pagetitle: This text is what will appear in the titlebar of the browser (blue titlebar on top of window)
  • Header: main-heading for page. This will appear in big font on top of the page
  • PostHeader: Optionally enter a post header. This will be smaller types below header
  • Description: Optionally enter a description of the page.
  • Body: This is the actual contents of the page. It can be formatted using the buttons above the text-area
  • When done with filling out the fields, hit the save button in the bottom of the page
  • This will save the document..
