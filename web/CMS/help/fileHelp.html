This is used for uploading/editing files. The files are used when you have pdf, word or other filetypes you want to link to. First you upload the file, next you create a document and select its format to be file, then you can select one of the files you uploaded.

File list

First screen is a list of the files.
  • To create a new file, simply hit the 'new' button.. This will take you to a screen where you can edit the properties of the new file.
  • To delete a file, mark it from the list, and hit the 'delete' button.
  • To edit a file, mark it from the list and hit the 'edit' button. This will take you to a screen where you can edit the properties of the selected file

Uploading/editing files

When hitting the edit or new button you will be redirected to a screen where you can edit the properties of the new file. If you are creating a new file, the fields will all be empty, else they will have the relevant values.

Below is a description of the fields and what they mean:

  • path: To upload a new image, simply hit the browse button to the right of this field, and select the image you want to upload.
  • identifier: This is a name, visible to you only.. Users of the homepage will NOT be able to see it. It is used to identify the files when you have them on a list, etc.
