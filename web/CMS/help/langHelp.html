This is used for adding or editing the available languages.

Language list

First screen is a list of the languages of the homepage.
  • To create a new language, simply hit the 'new' button.. This will take you to a screen where you can edit the properties of the new language.
  • To delete a language, mark it from the list, and hit the 'delete' button.
  • To edit a language, mark it from the list and hit the 'edit' button. This will take you to a screen where you can edit the properties of the selected language

Editing/adding languages

When hitting the edit or new button you will be redirected to a screen where you can edit the properties of the new language. If you are creating a new language, the fields will all be empty, if you are editing an existing language, the fields will have the relevant values.

Below is a description of the fields and what they mean

  • Name: This is the name of the language (eg. Danish).
  • Shorthand: This is the shorthand of the language. If the homepage is using mod rewrite, then this value will be a part of the address. (having 'dk' here on the homepage: www.yourdomain.com will make the address: dk.yourdomain.com when watching a document in danish).
  • Flag: This should be an image with the flag of the language you are editing. If you do not have the desired flag on this list, it can be uploaded via the 'Images' link in the left menu.
  • Priority: This is the priority of the languages. This dictates what order the choices of languages will be shown to the user. Furthermore, if a document does not exist in the translation the user desires, the translation of the desired document will default to the existing translation with the highest priority.
